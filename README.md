# Run this app

### `cd e-commerce-react` -> `npm install -g json-server`

### `npm install`

### `cd backend` -> `json-server products.json`

### `npm run dev`

# Clone this project

bash

`$ git clone https://github.com/Paiva2/e-commerce-react`

# Used Technologies

- HTML, CSS
- JavaScript, React
- React Router
- React Helmet
